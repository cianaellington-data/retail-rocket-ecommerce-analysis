# Retail Rocket E-commerce Capstone

This capstone analyzes **2.7M+ e-commerce events** from the Retail Rocket platform using **R** and the **tidyverse**.  
The goal is to explore user behavior, conversion trends, and top-performing products to identify opportunities for revenue growth and customer retention.

---

## Overview
The project examines:

- How users progress through the conversion funnel
- Hourly purchase patterns
- Top-performing categories and items
- Customer loyalty trends

---

## Key Analyses
- **Conversion Funnel:** View → Add to Cart → Transaction  
- **Purchases by Hour of Day**  
- **Top 10 Categories by Purchases**  
- **Top 10 Items by Purchases**  
- **Customer Loyalty Insights (Purchases per Customer)**  

---

## Files
- `Retail_Rocket_Capstone.ipynb` — Full Kaggle notebook with code and charts  
- `Retail_Rocket_Capstone.html` — Rendered version for easy viewing  
- `clean_events.csv` — Cleaned events dataset (subset of Kaggle dataset)  
- `clean_purchase_items.csv` — Cleaned purchase items dataset  

---

## Key Findings
- Only **2.6%** of users who view products add them to the cart; **0.84%** complete a purchase.  
- Purchases **peak between 6–8 PM**, highlighting the best marketing window.  
- Top 10 product categories contribute **~30% of revenue**.  
- Certain items show **high purchases per customer**, indicating strong loyalty.

---

## Tools & Libraries

## Dataset
This analysis uses anonymized data derived from the [Retail Rocket E-commerce dataset on Kaggle](https://www.kaggle.com/datasets/retailrocket/ecommerce-dataset).

---

## Recommendations
1. **Optimize Product Pages:** Improve product information, images, and calls-to-action to boost add-to-cart conversions.  
2. **Time Marketing Strategically:** Run campaigns during peak purchase hours (6–8 PM).  
3. **Prioritize Inventory & Promotions:** Focus on high-performing categories and items.  
4. **Drive Retention:** Target loyalty-driving items with personalized recommendations and repeat purchase incentives.
